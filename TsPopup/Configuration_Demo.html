划词弹窗演示

请选中这段文字中的任意部分，会在鼠标位置弹出操作弹窗，包含复制、搜索、翻译等功能。点击弹窗外的区域会自动关闭弹窗。

划词弹窗（TextSelectionPopup）是常见的网页交互功能，广泛应用于阅读类、文档类网站，提升用户操作体验。

你可以根据需求扩展更多功能按钮，比如分享、收藏、标注等，只需在弹窗中添加对应的按钮并实现相应逻辑即可。


Text Selection Popup Demo

Select any part of this text, and an action popup will appear at the mouse position, containing functions like copy, search, and translate. Clicking outside the popup will automatically close it.

The Text Selection Popup is a common web interaction feature, widely used on reading and document-related websites to enhance user experience.

You can expand it with more function buttons according to your needs, such as share, favorite, or annotate, simply by adding the corresponding buttons to the popup and implementing the related logic.
